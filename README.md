# MPFashions 👗🛍️

A sleek, responsive, single-page e-commerce web application built entirely with HTML, CSS (Bootstrap 5), and Vanilla JavaScript. 

MPFashions offers a seamless shopping experience where users can browse products, add them to their cart or wishlist, and manage their orders without the page ever reloading.

## ✨ Features

* **Single-Page Application (SPA):** Instant navigation between Home, Shop, Wishlist, Orders, About, and Contact sections using JavaScript DOM manipulation.
* **Dynamic Shopping Cart:** Add products, update quantities, calculate totals, and checkout smoothly.
* **Wishlist Management:** Save favorite items for later and easily move them to the shopping cart.
* **Order Tracking:** Keeps a history of previous checkouts with timestamps and cost breakdowns.
* **Responsive Design:** Fully mobile-friendly layout powered by Bootstrap 5.
* **Mock User Login & Contact Forms:** Interactive UI components for user engagement.

## 🛠️ Technologies Used

* **HTML5:** Semantic structuring of the web app.
* **CSS3 & Bootstrap 5:** For responsive grid layouts, modern typography, and clean UI components (modals, navbars, buttons).
* **Vanilla JavaScript (ES6):** Handles all the core logic (state management for the cart/wishlist, event listeners, and dynamic HTML rendering).
